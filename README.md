# Drupal React Component

## Prerequisites
* DDEV
* Node ^16

## Installation
* Run `ddev start` in the root folder
* Run `npm run build` in `/web/modules/custom/react_poc`
* Install Drupal using the existing config
* Create a `Landing page` node
* Create multiple `Product` nodes
